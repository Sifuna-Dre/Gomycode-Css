# GoMyCode Portfolio

Simple portfolio project with HTML and CSS. Contains an example bordered table and uses the Roboto font.

To push this project to GitHub:

1. Create a new repository on GitHub (do not initialize with a README).
2. Run the commands shown in the project root to add a remote and push.

Example:

```bash
git remote add origin https://github.com/USERNAME/REPO.git
git push -u origin main
```
